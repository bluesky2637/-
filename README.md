## 📚 关于本书

这是一本关于学习科学的经典著作，书中关于理工科学习方法、知识结构建立的论述在今天依然具有极高的参考价值。为了防止珍贵的资料因纸张老化而遗失，我将其制作成了数字版本。

- **书名**：怎样科学地学习 (同理工科大学生谈谈学习方法)
- **作者**：狄枚 (张笛梅)
- **出版年份**：1982 (第一版)
- **出版社**：北京出版社

## ✨ 项目特色

本项目简单的文字扫描，进行了深度的排版优化：

* **📄 文本识别**：使用 OCR 技术提取高精度文本，未校对修正。
* **🎨 排版优化**：使用 Markdown 重绘结构，适配 Obsidian 阅读流。
* **🖨️ A5 书籍复刻**：专门编写 CSS 样式，去除了传统打印的下划线，生成了拥有**宋体衬线字形**和**完美分页**的 A5 规格 PDF，获得接近实体书的阅读质感。
